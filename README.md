# CuentaBanco

Repositorio base para la actividad del curso **Construcción de Software**.

## Instrucciones
1. Realice un **fork** de este repositorio.
2. Implemente la clase `CuentaBanco` usando Programación Orientada a Objetos (POO).
3. Complete el archivo `app.py` agregando un menú interactivo desde la línea de comandos.
4. Use una **rama por funcionalidad** y fusione los cambios a `main`.
5. Cumpla con el estándar de codificación **PEP-8**.

## Funcionalidades requeridas
- Depósito
- Retiro
- Transferencia
- Consulta de saldo
